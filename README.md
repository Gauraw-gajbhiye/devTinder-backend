# DevTinder Backend (MERN Stack)

A backend system for a social matching application, built using Node.js, Express, and MongoDB.

## 🚀 Features
- User authentication using JWT
- Secure login with cookies
- User profile management
- Send and manage connection requests
- RESTful API design

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## 📁 Project Structure
src/
 ├── controllers/
 ├── models/
 ├── routes/
 ├── middleware/
 ├── config/

## ⚙️ Setup Instructions

1. Clone the repo
2. Install dependencies
   npm install

3. Create `.env` file
   DB_CONNECTION_SECRET=your_mongo_uri
   SECRET_KEY=your_secret

4. Run server
   npm run dev
